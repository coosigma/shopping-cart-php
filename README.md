# Installation

-   clone the project
-   composer install
-   set up a database and add the .env file

# Run dev server:

`php artisan serve --host 0.0.0.0`

# End-point:

`localhost:8000`

# Main files:

-   class file:
    `app/Cart.php`
-   route file:
    `routes/web.php`
-   template file:
    `resources/views/welcome.blade.php`
